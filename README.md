# como executar

```
# tab 1
git clone https://github.com/Edmar-Sousa/blog-samplemed.git

cd blog-samplemed/backend
./bin/cake migrations migrate
./bin/cake migrations seed --seed UsersSeed
./bin/cake server

composer install


# tab 2
cd blog-samplemed/fronend

npm install
npm run dev
```
